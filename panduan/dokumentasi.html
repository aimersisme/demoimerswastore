Dokumentasi Resmi iMersWAstore

Platform Pembuat Toko Online & POS Kasir Berbasis WhatsApp Gateway

Selamat datang di iMersWAstore! Platform revolusioner yang memungkinkan Anda membangun jaringan bisnis Software as a Service (SaaS) atau mengelola ekosistem toko online dan Point of Sale (POS) dalam satu pintu. Didukung oleh integrasi WhatsApp Gateway multi-provider.

Fitur Unggulan Sistem (Features)

iMersWAstore dirancang untuk ringan, super cepat, dan sepenuhnya responsif (Mobile-Friendly).

1 Manajemen Arsitektur Fleksibel (SaaS / Single Store)

  • ● SaaS Mode: Super Admin dapat membuat paket langganan (Pricing Plans), memanajemen user (Toko/Merchant), dan memberikan hak akses Agency kepada user untuk membuat toko turunannya sendiri.
  • ● Single Store Mode: Sistem beroperasi eksklusif untuk satu brand/toko tanpa fitur manajemen jaringan.

2 Toko Online Interaktif (Storefront)

  • ● Tampilan etalase elegan dan ramah seluler (mobile-friendly).
  • ● Fitur pencarian produk instan dan filter kategori.
  • ● Detail produk lengkap dengan dukungan Varian Fleksibel (warna, ukuran, harga beda per varian, stok spesifik varian).
  • ● Secret Tap Login: Keamanan ekstra bergaya easter-egg. Ketuk logo/teks di footer 5x untuk memunculkan form PIN rahasia akses admin.

3 Smart Checkout & Integrasi Ongkir

  • ● Sistem keranjang belanja dinamis.
  • ● Pemilihan area ongkos kirim manual yang terintegrasi langsung dengan Grand Total.
  • ● Fitur Kupon Diskon (berdasarkan Nominal Rp atau Persentase %).
  • ● Opsi pembayaran terpusat: Tunai/COD, Transfer Bank Manual, hingga integrasi QRIS Statis.

4 Kasir POS (Point of Sale) Offline

  • ● Modul kasir canggih di dalam Dashboard Admin untuk melayani pembeli walk-in (di tempat).
  • ● Sistem kalkulator otomatis (Uang Diterima & Kembalian).

Panduan Integrasi URL (Custom Landing Page)

Fitur ini didedikasikan untuk Anda atau Mitra (Agency) yang ingin menggunakan desain Landing Page buatan sendiri (menggunakan WordPress, Sejoli, Berdu, Webflow, dll) namun tetap ingin datanya masuk ke mesin iMersWAstore.

Anda tidak perlu menyentuh kode sistem. Cukup arahkan tombol-tombol di Landing Page buatan Anda ke 3 Rute URL Integrasi berikut:

(Asumsi domain sistem Anda adalah: https://app.domainanda.com)

3 Rute URL Integrasi Utama

1. Link Menuju Form "Daftar Sekarang / Register"

Jika pengunjung mengklik tombol langganan/daftar di Landing Page custom Anda, arahkan mereka ke URL ini agar form pop-up pendaftaran langsung otomatis terbuka:

Format URL:
https://app.domainanda.com/index.html?register=true
Contoh HTML:
<a href="https://app.domainanda.com/index.html?register=true">Daftar Sekarang</a>

2. Link Menuju "Login Dashboard"

Arahkan tombol "Masuk / Login Admin" di website Anda ke URL ini:

Format URL:
https://app.domainanda.com/login.html

3. Link Menuju Etalase Toko (Storefront)

Berdasarkan sistem URL Rewrite (seperti di Vercel), setiap toko yang berhasil dibuat akan memiliki slug atau nama URL unik. Format untuk membuka toko tersebut adalah:

Format URL:
https://app.domainanda.com/{slug-toko}
Contoh Penggunaan:

Jika user membuat toko bernama "Kopi Senja" dengan slug kopi-senja, maka link tokonya adalah:

https://app.domainanda.com/kopi-senja

Catatan: Sistem secara cerdas akan membaca path URL tersebut dan otomatis memuat katalog produk milik toko "Kopi Senja".
